What is SIMPLE_SQL?

SIMPLE_SQL is a comprehensive SQLite library for Eiffel that provides:

  • Fluent Query Builders - Chainable, type-safe query construction
  • Prepared Statements - Parameterized queries for security and performance
  • Streaming Results - Memory-efficient cursor and callback-based iteration
  • Advanced Features - FTS5, JSON1, audit trails, vector embeddings
  • Developer Experience - Schema introspection, migrations, error handling

Quick Example

local
    db: SIMPLE_SQL_DATABASE
    result: SIMPLE_SQL_RESULT
do
    -- Open database
    create db.make ("myapp.db")

    -- Fluent query builder
    result := db.select_from ("users")
        .columns ("id, name, email")
        .where ("active = 1")
        .order_by ("name")
        .limit (10)
        .execute

    -- Process results
    across result as row loop
        print (row.string_value ("name") + "%N")
    end

    db.close
end
